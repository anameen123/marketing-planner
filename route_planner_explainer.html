🗺️ Route Planner — Random clinic scatter + corridor levels

Member sets Start and End + how many stops they want in between (here: 5). The straight dashed line gets split into 5 equal slices. Each slice becomes a rectangle extending width mi each side. Clinics/MVAs anywhere on the map are bucketed into the level whose rectangle they fall inside. Anything outside the corridor is ignored unless the user expands the width slider.

What the picture shows: 5 stops requested between Start and End. The dashed line is split into 5 equal slices of length d. Each slice becomes a corridor extended width mi on each side. Random clinic / MVA pins land everywhere on the map — only the ones inside the rectangles count. Faded pins above and below are ignored unless the user expands width.

🎨 Pin colors

Green · > 3 weeks since last visit (or never) — TOP PRIORITY
Orange · 2–3 weeks since last visit — secondary
Red · < 2 weeks ago — EXCLUDED from recs
Purple S/E · Start & End pins

📐 The math

L = straight-line(S → E)
d = L / N (here N = 5)
width = user-tunable (1 mi/side default)
in_level_i(pin):
  projection onto line ∈ [i·d, (i+1)·d]
  AND ⊥ distance < width
deroute(pin) = ⊥ distance from line

🎛️ Member controls

  • Set start / end via list pick OR map double-click
  • Number of stops (changes N, slices the line differently)
  • Corridor width slider — expand to pick up pins farther from the line
  • Pick 0–any candidates per level (some levels can stay empty)
